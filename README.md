# Dotfiles

## Installation

1. Clone the repository
2. Install homebrew from https://brew.sh/
3. Install Oh-my-zsh from https://ohmyz.sh/#install
4. Install stow `brew install stow`
5. Stow needed parts:

```bash
stow zsh
```
