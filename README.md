# Dotfiles

## Installation

1. Clone the repository
1. Install homebrew from https://brew.sh/
1. Install Oh-my-zsh from https://ohmyz.sh/#install
1. Install stow `brew install stow`

### Install needed instruments

```
brew install zellij
brew install tmux
brew install --cask alacritty
brew install neovim
```

### Stow needed parts:

```bash
stow zsh
```
