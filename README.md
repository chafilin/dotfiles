# Dotfiles

## Installation

1. Clone the repository
1. Install homebrew from https://brew.sh/
1. Install Oh-my-zsh from https://ohmyz.sh/#install
1. Install needed instruments: `brew bundle install --file ./Brewfile`
1. Stow needed parts: `stow zsh`
